QUICK START - BUILD APK IN 5 STEPS:
-----------------------------------

1. EXTRACT THE ZIP
   - Extract adaptalyfe-android-build.zip to your Mac
   - You'll get a folder: adaptalyfe-android-build/

2. OPEN TERMINAL
   cd /path/to/adaptalyfe-android-build

3. SYNC WEB ASSETS (one-time setup)

   npx cap sync android

4. BUILD APK
   cd android
   chmod +x gradlew

TROUBLESHOOTING:
---------------

Error: "SDK location not found"
Fix: Create android/local.properties with:
     sdk.dir=/Users/YOUR_USERNAME/Library/Android/sdk

Error: "Gradle build failed"
Fix: cd android
     ./gradlew clean
     ./gradlew assembleDebug

Error: "Permission denied: gradlew"
Fix: chmod +x android/gradlew
